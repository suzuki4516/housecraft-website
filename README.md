## 🚀 使用方法

### 1. ローカルで開く

ファイルをダウンロード後、`index.html` をブラウザで開いてください。

```bash
# シンプルなHTTPサーバーを起動する場合（推奨）
# Python 3の場合
python -m http.server 8000

# Node.jsのhttp-serverを使う場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス

### 2. 画像の差し替え

現在はUnsplashのプレースホルダー画像を使用しています。
実際の施工事例写真に差し替える場合：

1. `index.html` 内の `https://images.unsplash.com/...` の部分を実際の画像パスに変更
2. 画像ファイルは `images/` フォルダに配置することを推奨

例：
```html
<!-- 変更前 -->
<img src="https://images.unsplash.com/photo-xxx" alt="...">

<!-- 変更後 -->
<img src="images/project-01.jpg" alt="...">
```

## 🔧 カスタマイズ方法

### テキストの変更
`index.html` 内のテキストを直接編集してください。

### 色の変更
`style.css` の `:root` セクションでカラー変数を変更できます。

```css
:root {
    --color-primary: #YOUR_COLOR;
    /* ... */
}
```

### アニメーション速度の調整
`script.js` 内の各クラスで調整可能です。

```javascript
// スライダーの速度（ミリ秒）
this.slideInterval = setInterval(() => {
    this.nextSlide();
}, 6000); // 6秒 → お好みの秒数に変更
```
